# 📚 Exercícios de Desenvolvimento Web

Este repositório reúne os códigos e exercícios desenvolvidos durante as aulas de **Desenvolvimento Web**.  
O objetivo é praticar conceitos fundamentais de **HTML, CSS e JavaScript**, além de acompanhar minha evolução ao longo da disciplina.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 📂 Estrutura do Repositório

Os arquivos estão organizados por aula ou tema estudado.

## 🎯 Objetivo

- Praticar conceitos de desenvolvimento web
- Aprender estruturação de páginas com HTML
- Estilização com CSS
- Lógica e interatividade com JavaScript
- Registrar minha evolução no aprendizado
